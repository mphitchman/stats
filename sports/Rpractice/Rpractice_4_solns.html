(After the loop ends, results should have 10000 values.) Then report what the mean and standard deviation of the results vector. Also make a histogram of the results to see what the shape looks like.

results <- c()

for (i in 1:10000){
  A = sample(0:9,1,replace=FALSE)
  B = sample(0:9,1,replace=FALSE)
  results[i] <- 10*A + B
}

barplot(table(results))

Part IV: creating functions

skeeball_game <- function(px = c(.1,.2,.1,.1,.4,.1), n = 6){
    #px = probability distribution for X
    #n = number of balls
    score <- sum(sample(seq(0,50,10),
                        n,
                        replace=T,
                        prob=px))
    return(score)
}

12. Write a for loop that plays 10000 games between myself (mph) and my brother Dave and records each of our scores in each game as well as how many times each of us won. When the dust settles, who won more often? How many times did we tie?

# results vectors for my scores and Dave's
mph = c() 
dave = c() 

# counters for my wins and Dave's
mph_wins = 0 
dave_wins = 0 

for (i in 1:10000){
  mph[i] = skeeball_game(px = c(.1,.2,.1,.1,.4,.1),n=6)
  dave[i] = skeeball_game(px = c(.1,.2,.2,.1,.1,.3),n=6)
  
  if (mph[i] > dave[i]){mph_wins = mph_wins + 1}
  if (mph[i] < dave[i]){dave_wins = dave_wins + 1}
}
cat(mph_wins,dave_wins)
## 4664 4656

I won 4664 times; Dave won 4656 times, and so we must have tied 680 times.

13. Write a function called chuck_a_luck that takes one input variable k, rolls three fair 6-sided dice and returns how many times the value k appears on the three dice. For instance, if the input variable is k = 3, and you roll 4, 4, and 6, the function should return 0.

chuck_a_luck <- function(k){
  x = sample(1:6,3,replace=TRUE)
  return(sum(x == k))
}
